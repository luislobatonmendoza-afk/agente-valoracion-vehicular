# 🏛️ Implementación de un Agente Inteligente de Clasificación para la Valoración de Activos Vehiculares

Este repositorio contiene el código fuente y los artefactos del proyecto final desarrollado para el curso **Inteligencia Artificial: Principios y Técnicas** (Semestre 2026-1) en la **Universidad Privada Antenor Orrego (UPAO)**.

## 📝 Descripción del Proyecto
El sistema resuelve la asimetría de información en el mercado secundario de vehículos mediante un **Agente Inteligente de Clasificación**. Utilizando algoritmos de Machine Learning (Random Forest, SVM y MLP), el agente procesa variables técnicas y comerciales del activo para categorizarlo en tres rangos de valor objetivos: **Económico**, **Estándar** o **Premium**.

## 📁 Estructura del Repositorio
* `app.py`: Script principal de la aplicación interactiva construida con Gradio.
* `requirements.txt`: Lista de dependencias del entorno de ejecución.
* `rf_pipeline.pkl`: Serialización del pipeline de preprocesamiento y modelo clasificador.
